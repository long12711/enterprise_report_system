   - 专家用户（高级专家、中级专家、初级专家）

✅ 完整的API支持

   - 提交问卷时保存用户类型和分级信息

   - 直观的用户类型和分级选择界面

🔧 技术实现
================================================================================

用户类型和分级体系:
  - 3个用户类型
  - 9个分级 (3×3)
  - 9个问卷配置

API端点:
  - GET /api/user-types               - 获取用户类型和分级
  - GET /api/get_questions            - 获取问卷题目 (支持过滤)
  - POST /api/submit_questionnaire    - 提交问卷

前端功能:
  - 用户类型选择下拉菜单
  - 用户分级选择下拉菜单 (动态更新)
  - 问卷说明信息框
  - 动态问卷题目加载

后端功能:
  - 用户类型和分级管理
  - 问卷题目过滤
  - 数据保存和验证
  - 报告生成

🚀 快速开始
================================================================================

1. 启动系统
   cd d:\Claude\ Code\enterprise_report_system
   python app.py

2. 访问问卷页面
   http://localhost:5000/questionnaire

3. 选择用户类型和分级
   - 选择用户类型 (工商联/企业/专家)
   - 选择分级
   - 系统自动加载对应的问卷题目

4. 填写问卷并提交
   - 填写企业信息
   - 回答问卷题目
   - 点击提交

5. 查看报告
   - 报告将通过邮件发送
