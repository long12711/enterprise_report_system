## 🎨 格式说明

### 标题样式
- **主标题**：22号黑体，蓝色（RGB: 46, 80, 144）
- **副标题**：16号楷体，蓝色
- **一级指标**：13号黑体，蓝色
- **二级指标**：11号黑体

### 正文样式
- **题目**：10.5号宋体
- **选项**：10.5号宋体，带复选框
- **说明文字**：9号宋体，斜体，灰色

### 页面设置
- **页边距**：上下1英寸，左右1.25英寸
- **字体**：中文宋体，英文Times New Roman

## 💡 使用方法

### 方法1：直接使用生成的文件
1. 打开生成的Word文档
2. 填写企业信息
3. 在相应选项前的□打√
4. 填写详细说明
5. 签字并保存

### 方法2：重新生成
```bash
# 运行生成脚本
python generate_word_questionnaire_template.py
```

### 方法3：自定义生成
```python
from generate_word_questionnaire_template import WordQuestionnaireGenerator

# 创建生成器
generator = WordQuestionnaireGenerator('nankai_indicators.xlsx')

# 生成特定级别
generator.generate_questionnaire('初级')

# 或生成所有级别
generator.generate_all_levels()
```
