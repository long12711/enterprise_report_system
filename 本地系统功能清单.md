## 1️⃣ 南开问卷系统 ⭐ 核心功能

### 问卷生成
```
✅ 初级问卷 (65题) - 基础指标
✅ 中级问卷 (52题) - 鼓励指标  
✅ 高级问卷 (42题) - 拔高指标

支持格式:
- Word (.docx) - 适合打印
- Excel (.xlsx) - 适合数据处理
```

### 在线填写
```
访问: http://localhost:5000/nankai-survey

功能:
✅ 选择级别
✅ 填写企业信息（6个字段）
✅ 逐题作答（单选 + 部分完成说明）
✅ 上传佐证材料
✅ 提交后立即显示得分
```

### 智能评分
```
评分类型:
✅ 二元评分 (是/否)
✅ 全部完成评分 (完成/部分/未完成)
✅ 多项评分 (完成X项得分)
✅ 负向评分 (一票否决)
✅ 分级评分 (0-2分范围)

输出:
- 实际得分 (如: 45.6分)
- 满分 (如: 100分)
- 百分制得分 (如: 45.6%)
- 详细得分明细
```

### 数据下载
```
格式: Excel (.xlsx)

包含:
✅ 企业基本信息
✅ 所有题目和答案
✅ 部分完成说明
✅ 得分明细
✅ 评分标准
```

### 自评报告
```
格式: Word (.docx)

内容:
✅ 封面（企业名称、时间）
✅ 执行摘要（得分、等级）
✅ 九大维度分析
✅ 改进建议
✅ 附录（详细得分表）

生成: 提交后自动生成并邮件发送
```

## 4️⃣ 等级升级机制

### 企业等级
```
初级 → 中级 → 高级

升级条件:
- 初级→中级: 得分 ≥ 60分
- 中级→高级: 得分 ≥ 60分

升级方式:
1. 自动判定（达到分数线）
2. 提交专项申请
3. 工商联审核批准
```
