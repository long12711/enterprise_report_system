## 🎨 格式规范说明

### 1. 文档样式

**正文**:
- 字体：仿宋_GB2312
- 字号：四号（16pt）
- 行距：1.5倍
- 首行缩进：2字符（0.5英寸）

**一级标题**:
- 字体：黑体
- 字号：二号（22pt）
- 对齐：居中
- 格式：一、二、三...

**二级标题**:
- 字体：黑体
- 字号：三号（18pt）
- 格式：（一）、（二）、（三）...

**三级标题**:
- 字体：黑体
- 字号：四号（16pt）

### 2. 表格样式

- 样式：Light Grid Accent 1
- 表头：加粗，字号13pt
- 内容：仿宋_GB2312，字号12pt
- 对齐：居中
- 行高：0.35英寸

### 3. 特殊标记

**亮点标记**:
- 格式：【主要亮点】
- 颜色：蓝色（RGB: 0, 102, 204）
- 字体：加粗

### 4. 叙述性语言特点

**避免直白数据**:
- ❌ 不说："得分92.3%"
- ✅ 而说："制度建设成效显著，制度体系健全完善"

**委婉表述问题**:
- ❌ 不说："制度建设差"
- ✅ 而说："对标高质量发展要求，部分领域仍有提升空间"

**战略性建议**:
- ❌ 不说："赶紧建立制度"
- ✅ 而说："系统谋划现代制度建设工作，坚持问题导向和目标导向相结合"

## 📊 内容生成逻辑

### 1. 企业定位自动判定

```python
if 年收入 > 50000万 or 员工 > 500人:
    定位 = "行业领军企业"
elif 年收入 > 10000万 or 员工 > 200人:
    定位 = "区域骨干企业"
else:
    定位 = "成长型企业"
```

### 2. 水平描述自动生成

```python
if 得分率 >= 90%:
    描述 = "制度建设成效显著"
elif 得分率 >= 80%:
    描述 = "制度体系较为完善"
elif 得分率 >= 70%:
    描述 = "制度建设总体良好"
elif 得分率 >= 60%:
    描述 = "基本建立了相关制度"
else:
    描述 = "制度建设有待加强"
```

### 3. 改进优先级判定

```python
if 得分率 < 40%:
    优先级 = "高优先级"
elif 得分率 < 50%:
    优先级 = "较高优先级"
else:
    优先级 = "中等优先级"
```
