# 工商联门户菜单结构图

## 菜单层级关系

```
工商联门户
│
├─ 企业管理
│  └─ 资质审核 (→ 企业资质审核)
│     └─ 企业资质审核
│        - 根据企业问卷得分与指标，判断是否符合升级条件
│        - 支持按审核级别筛选
│        - 支持批准升级和手动晋级操作
│
├─ 专家管理
│  └─ 专家辅导记录 (→ 辅导记录查看)
│     └─ 辅导记录查看
│        - 查看专家与企业之间的沟通记录
│        - 显示专家、企业、消息内容和时间
│
├─ 工商联用户管理
│  └─ 用户管理 (→ 工商联用户列表)
│     └─ 工商联用户列表
│        - 管理工商联系统的用户账户和权限
│        - 支持编辑和删除用户
│        - 显示用户名、邮箱、角色、审核级别等信息
│
├─ 专项反馈管理
│  └─ 专项申请审核 (→ 专项申请审核)
│     └─ 专项申请审核
│        - 审核企业提交的专项申请
│        - 支持查看附件
│        - 支持通过/驳回操作
│        - 支持添加备注
│
├─ 基础问卷管理
│  └─ 问卷管理 (→ 问卷配置)
│     └─ 问卷配置
│        - 管理和配置系统中的基础问卷
│        - 支持创建、编辑、预览和删除问卷
│        - 显示问卷名称、问题数量、创建时间和状态
│
└─ 平台基础功能管理
   └─ 报告管理 (→ 查看企业报告 / 发送报告)
      ├─ 查看企业报告
      │  - 查看系统内已生成的所有企业评估报告
      │  - 支持下载报告
      │
      └─ 发送报告
         - 选择一份报告并将其通过邮件发送给指定收件人
         - 支持输入收件人邮箱
```

## 悬浮窗口交互流程

```
用户操作流程：

1. 用户进入工商联门户
   ↓
2. 左侧显示6个主要功能分类
   ├─ 企业管理
   ├─ 专家管理
   ├─ 工商联用户管理
   ├─ 专项反馈管理
   ├─ 基础问卷管理
   └─ 平台基础功能管理
   ↓
3. 用户将鼠标悬停在某个菜单项上
   ↓
4. 右侧弹出悬浮窗口，显示该菜单项的二级功能
   ↓
5. 用户点击二级菜单中的某一项
   ↓
6. 页面切换到对应的功能模块
   ↓
7. 加载相应的数据和界面
```

## 样式特点

### 菜单分组
- 每个分组有清晰的标题标签（大写、灰色、字体较小）
- 分组之间有适当的间距
- 分组内的菜单项左对齐

### 菜单项
- 菜单项文本左对齐，箭头符号右对齐
- 鼠标悬停时背景变深
- 当前活跃的菜单项左边框变蓝色
- 菜单项右侧显示 "→" 箭头指示符

### 悬浮窗口
- 位置：菜单项右侧（距离240px）
- 背景色：深灰色 (#1f2937)
- 边框：浅灰色 (#374151)
- 圆角：8px
- 最小宽度：180px
- 阴影：0 10px 25px rgba(0,0,0,0.3)
- 动画：0.2s 淡入淡出

### 二级菜单项
- 文字颜色：浅灰色 (#cbd5e1)
- 字体大小：13px
- 鼠标悬停时：背景变深灰色，文字变白
- 内边距：10px 16px

## 响应式设计考虑

当前版本为桌面版设计。后续可考虑添加：

1. **平板设备**：菜单可折叠为图标模式
2. **移动设备**：菜单改为抽屉式导航
3. **小屏幕**：悬浮窗口改为下拉菜单

## 性能优化

- 使用事件委托处理菜单点击
- 悬浮窗口使用 CSS 过渡而非 JavaScript 动画
- 数据按需加载（点击时才加载）
- 使用 Flexbox 布局，性能优于 Grid

## 无障碍设计

建议添加：
- ARIA 标签用于屏幕阅读器
- 键盘导航支持（Tab 键切换菜单项）
- 高对比度模式支持
- 焦点指示器
