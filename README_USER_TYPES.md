## 🔗 快速链接

### 系统访问

- 问卷页面: `http://localhost:5000/questionnaire`
- 管理员登录: `http://localhost:5000/admin/login`
- API基础URL: `http://localhost:5000/api`

### API端点

- 获取用户类型: `GET /api/user-types`
- 获取问卷题目: `GET /api/get_questions`
- 提交问卷: `POST /api/submit_questionnaire`

### 配置文件

- 用户类型配置: `user_types_config.py`
- 问卷数据: `指标体系.xlsx`
- 前端页面: `templates/questionnaire.html`
- 后端应用: `app.py`
