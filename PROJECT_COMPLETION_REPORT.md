## 🎯 项目目标

### 主要目标

实现一个支持三类用户（工商联、企业、专家）各自分级体系的问卷系统，根据用户类型和分级动态加载不同的问卷题目。

### 次要目标

- 提供完整的API接口
- 编写详细的文档
- 确保系统的可扩展性和可维护性
- 保持向后兼容性

## 🔧 技术实现

### 架构设计

```
前端 (Frontend)
├── HTML (用户类型和分级选择)
├── CSS (样式)
└── JavaScript (动态加载和交互)

后端 (Backend)
├── Flask应用 (app.py)
├── 配置模块 (user_types_config.py)
├── API端点 (3个)
└── 业务逻辑 (题目过滤、数据保存)

数据 (Data)
├── 指标体系 (Excel文件)
├── 用户类型配置 (Python配置)
└── 提交数据 (JSON文件)
```

### 核心功能

1. **用户类型和分级管理**
   - 定义了三类用户和分级体系
   - 支持灵活的配置和扩展

2. **动态问卷加载**
   - 根据用户类型和分级过滤题目
   - 支持多种过滤规则

3. **API接口**
   - 获取用户类型和分级
   - 获取问卷题目
   - 提交问卷

4. **前端交互**
   - 用户类型选择
   - 分级选择
   - 问卷说明
   - 动态题目加载
