## 🎯 功能亮点

### 客户端
1. **渐变色UI** - 紫色渐变背景，视觉效果现代
2. **拖拽上传** - 支持文件拖拽，提升用户体验
3. **进度保存** - 问卷自动保存到浏览器，防止数据丢失
4. **实时预览** - 上传后立即预览数据，确认无误再处理
5. **响应式设计** - 自适应各种屏幕尺寸

### 管理端
1. **侧边导航** - 深蓝色侧边栏，专业后台风格
2. **统计卡片** - 悬浮动画效果，数据一目了然
3. **批量操作** - 高效处理大量数据
4. **实时监控** - 批量任务实时进度显示
5. **搜索过滤** - 快速定位目标数据
6. **模态框** - 详情展示不跳转，操作流畅

## 🎨 设计规范

### 颜色方案

#### 客户端
- **主色调**: #667eea → #764ba2 (紫色渐变)
- **背景色**: #f8f9ff (淡蓝紫)
- **文字色**: #333 (深灰)
- **辅助色**: #666 (中灰)

#### 管理端
- **侧边栏**: #2E5090 → #1a3a6b (深蓝渐变)
- **主色调**: #667eea (紫蓝)
- **成功色**: #28a745 (绿色)
- **警告色**: #ffc107 (黄色)
- **危险色**: #dc3545 (红色)
- **信息色**: #17a2b8 (青色)

### 字体规范
- **标题**: Microsoft YaHei, 微软雅黑
- **正文**: Microsoft YaHei, Arial, sans-serif
- **代码**: Courier New, monospace

### 尺寸规范
- **大标题**: 32px
- **中标题**: 24px
- **小标题**: 20px
- **正文**: 14-16px
- **辅助文字**: 12px

### 圆角规范
- **卡片**: 10-20px
- **按钮**: 5-25px
- **输入框**: 5-8px

### 阴影规范
- **卡片阴影**: 0 2px 10px rgba(0,0,0,0.05)
- **悬浮阴影**: 0 5px 20px rgba(0,0,0,0.1)
- **按钮阴影**: 0 4px 15px rgba(102,126,234,0.4)

## 📱 响应式设计

### 断点设置
- **桌面端**: > 1024px
- **平板端**: 768px - 1024px
- **手机端**: < 768px

### 适配策略
- 使用CSS Grid和Flexbox布局
- 统计卡片自动换行
- 表格横向滚动
- 侧边栏在移动端可收起

## 🚀 性能优化

1. **异步加载** - 批量任务后台处理
2. **分页显示** - 大数据量分页展示
3. **懒加载** - 图片和报告按需加载
4. **缓存策略** - 静态资源浏览器缓存
5. **压缩传输** - 生产环境启用Gzip

## 🔒 安全措施

1. **Session认证** - 管理员登录状态验证
2. **密码加密** - SHA256哈希存储
3. **文件验证** - 上传文件类型和大小限制
4. **XSS防护** - 用户输入转义
5. **CSRF防护** - Flask SECRET_KEY

## 📊 数据流程

### 在线问卷流程
```
用户填写问卷
    ↓
提交到后端 /api/submit_questionnaire
    ↓
保存JSON到 submissions/
    ↓
生成Excel文件
    ↓
异步生成Word和PDF报告
    ↓
发送邮件通知
```

### Excel上传流程
```
上传Excel文件
    ↓
后端验证格式 /upload
    ↓
返回预览数据
    ↓
用户确认并配置选项
    ↓
后端处理 /process
    ↓
逐行生成报告
    ↓
发送邮件/短信通知
    ↓
返回处理结果
```

### 管理员批量生成流程
```
选择提交记录
    ↓
配置报告选项
    ↓
提交到 /api/admin/batch-generate
    ↓
创建异步任务
    ↓
返回任务ID
    ↓
前端轮询 /api/admin/batch-status
    ↓
显示实时进度
    ↓
完成后提示
```
