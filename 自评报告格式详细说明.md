# 南开问卷自评报告格式详细说明

## 报告概述

自评报告是企业填写南开问卷后自动生成的专业评价文档，采用Word格式（.docx），包含7个主要章节。

## 报告结构

### 📄 完整目录

```
南开大学现代企业制度指数评价自评报告
├── 封面
├── 一、企业基本信息
├── 二、评价总览
├── 三、各维度详细分析
├── 四、优势与不足
├── 五、改进建议
├── 六、结语
└── 七、附录：问卷详细数据
```

---

## 详细格式说明

### 1. 封面

**样式**：居中对齐，蓝色主题

```
═══════════════════════════════════════

    南开大学现代企业制度指数
         评价自评报告

═══════════════════════════════════════

        企业名称：XX科技有限公司

        评价级别：初级

        综合得分：85.50分

        生成日期：2025年12月01日

═══════════════════════════════════════
```

**格式细节**：
- 标题：22号字，加粗，深蓝色(RGB: 46, 80, 144)
- 企业名称：16号字，加粗
- 评价级别：14号字
- 综合得分：18号字，加粗，深蓝色
- 生成日期：12号字

### 3. 二、评价总览

**格式**：段落文本 + 评级说明

```
二、评价总览
═══════════════════════════════════════

评价结果：贵企业在初级问卷评价中，实际得分70.50分，
满分85.00分，百分制得分为82.94分。

评价等级：良好

总体评价：企业现代制度建设较为完善，主要制度体系健全，
执行情况良好。
```

**评级标准**：

| 百分制得分 | 评价等级 | 说明 |
|-----------|---------|------|
| ≥90分 | 优秀 | 企业现代制度建设达到卓越水平，各项制度健全完善，执行有力 |
| 80-89分 | 良好 | 企业现代制度建设较为完善，主要制度体系健全，执行情况良好 |
| 70-79分 | 中等 | 企业现代制度建设基本健全，部分制度需要进一步完善 |
| 60-69分 | 及格 | 企业现代制度建设有待提升，需要加强制度建设和执行力度 |
| <60分 | 需改进 | 企业现代制度建设亟需加强，建议系统性地完善各项制度 |

**格式细节**：
- 标题：一级标题，深蓝色
- 评价等级：加粗，深蓝色
- 段落：12号字，宋体

---

### 4. 三、各维度详细分析

**格式**：按一级指标分组，每个维度包含得分情况和详细题目

```
三、各维度详细分析
═══════════════════════════════════════

【党建】
─────────────────────────────────────

维度得分：8.50分 / 10.00分 (85.00%)

完成情况：
✓ 规范党组织建设 (1.00/1.00分)
  答案：A. 是（已建立/制定/设立）
  
✓ 执行党的组织生活制度 (1.00/1.00分)
  答案：A. 全部完成
  
○ 践行社会主义核心价值观体系建设 (1.50/2.00分)
  答案：B. 完成1-2项
  说明：已完成企业文化建设制度融入和员工认同率调查

─────────────────────────────────────

【股权结构】
─────────────────────────────────────

维度得分：12.00分 / 15.00分 (80.00%)

完成情况：
✓ 股权结构清晰 (2.00/2.00分)
  答案：A. 完全达到要求
  
✓ 股东权利保护 (2.00/2.00分)
  答案：A. 是
  
○ 股权激励机制 (1.50/2.00分)
  答案：B. 部分完成
  说明：已建立股权激励制度，正在实施中

...（其他维度）
```

**符号说明**：
- ✓ 完全得分（得分率100%）
- ○ 部分得分（得分率50%-99%）
- ✗ 未得分（得分率<50%）

**格式细节**：
- 一级标题：深蓝色
- 二级标题（维度名）：14号字，加粗
- 维度得分：加粗显示
- 题目列表：缩进，带符号
- 答案：斜体
- 说明：灰色，小字

### 5. 四、优势与不足

**格式**：分两部分，列举主要优势和不足

```
四、优势与不足
═══════════════════════════════════════

【主要优势】

根据评价结果，贵企业在以下方面表现突出：

1. 党建工作扎实
   - 党组织建设规范，制度健全
   - 组织生活制度执行到位
   - 得分率：85.00%

2. 股权结构清晰
   - 股权关系明确，权责清晰
   - 股东权利得到有效保护
   - 得分率：80.00%

3. 治理结构完善
   - 三会一层运作规范
   - 决策程序科学合理
   - 得分率：88.00%

【需要改进的方面】

以下方面需要进一步加强：

1. 风险管理体系
   - 风险识别机制需要完善
   - 风险应对措施不够系统
   - 得分率：60.00%
   - 建议：建立全面风险管理体系

2. 信息披露制度
   - 信息披露不够及时
   - 透明度有待提高
   - 得分率：55.00%
   - 建议：完善信息披露制度和流程

3. 社会责任履行
   - 社会责任意识需要加强
   - 公益活动参与度较低
   - 得分率：50.00%
   - 建议：制定社会责任战略规划
```

**筛选标准**：
- 优势：得分率≥80%的维度
- 不足：得分率<70%的维度

**格式细节**：
- 标题：一级标题，深蓝色
- 子标题：【】包围，加粗
- 列表：编号列表
- 得分率：加粗显示
- 建议：缩进，斜体

### 6. 五、改进建议

**格式**：针对性建议，分短期和长期

```
五、改进建议
═══════════════════════════════════════

基于评价结果，我们提出以下改进建议：

【短期改进措施（1-3个月）】

1. 完善风险管理制度
   - 建立风险识别和评估机制
   - 制定风险应对预案
   - 明确风险管理责任人

2. 加强信息披露
   - 制定信息披露管理办法
   - 建立信息披露审核流程
   - 定期发布企业信息

3. 提升员工培训
   - 开展制度培训
   - 加强合规意识教育
   - 建立培训考核机制

【中长期改进措施（3-12个月）】

1. 建立全面风险管理体系
   - 引入专业风险管理工具
   - 建立风险管理信息系统
   - 定期开展风险评估

2. 完善公司治理结构
   - 优化董事会结构
   - 加强独立董事作用
   - 完善监事会职能

3. 加强社会责任建设
   - 制定社会责任战略
   - 开展公益活动
   - 建立利益相关方沟通机制

【持续改进建议】

1. 定期开展自评
   - 建议每年进行一次全面自评
   - 跟踪改进措施落实情况
   - 持续优化制度体系

2. 学习标杆企业
   - 研究行业最佳实践
   - 参加专业培训和交流
   - 引入外部专家指导

3. 建立持续改进机制
   - 设立制度建设专项小组
   - 建立制度评估和更新机制
   - 营造制度文化氛围
```

**格式细节**：
- 标题：一级标题，深蓝色
- 时间分类：【】包围，加粗
- 建议列表：编号列表，多级缩进
- 重点内容：加粗或下划线

## 报告生成流程

```
用户提交问卷
    ↓
系统保存提交数据（JSON格式）
    ↓
调用评分引擎计算得分
    ↓
调用报告生成器
    ↓
读取指标体系和提交数据
    ↓
生成7个章节内容
    ↓
保存为Word文档
    ↓
返回报告文件路径
```

## 报告文件信息

### 文件命名
```
自评报告_{企业名称}_{时间戳}.docx
```

**示例**：
```
自评报告_XX科技有限公司_20251201_235959.docx
```

### 存储位置
```
storage/nankai_reports/
```

### 文件大小
- 典型大小：50-200 KB
- 取决于：题目数量、答案详细程度
